## 三. SDK集成方法介绍   
### 3.1 系统要求    
* 最低支持Android版本：**minSdkVersion 21**
* 支持CPU架构： **armeabi-v7a 、 arm64-v8a**

### 3.2 下载工程
复制工程中libs目录下的aar库，具体参见DEMO    


### 3.3 集成时的注意事项
* **本SDK基于Android X**
* **注意音视频权限获取**
